# Optik Tack (Abschluss Projekt)

Hier findest du meine Webseite, welche ich als Abschlussprojekt für meine Designer-Ausbildung abgeben habe.
Ursprünglisch wurde diese Seite mit Html,PHP und JQuery entwickelt - wird hier aber mal etwas aufgefrischt, damit sie nach all der Zeit noch richtig dargestellt werden kann.
Zudem kommen paar kleine Optimierungen dazu.

Die Webseite ist nur für Desktop entwickelt und funktioniert daher auf Mobile nicht.
Im Branch "Release/V1.0-Original" findest du noch das original abgegebene Projekt, wobei auch dieses bereits über das HTML läuft, damit die Git-Pages funktionieren.
Ein Abschicken des Formulars ist als nur über ein locales Hosting wie z.B. XAMP möglich. (Oder online Hoster)

---

## Zur Aufgabe
Hierbei ging es darum für ein Fiktives Optiker-Unternehmen eine Webseite zu gestalten, die möglichst ansprechend für den kunden ist, die Firma als modern representiert und gleichzeitig dazu einlädt, den Optiker zu besuchen.
Gegeben waren Texte und Bilder, die nach Belieben gestaltet werden konnten.
Das Konzept wie Entwicklung der Seite wurde uns überlassen.
Es wurden auch Hilfen wie Bootstrap erlaubt.
Die Abgabe erfolgte zusammen mit allen anderen Materialien auf eine selbstgestaltete CD.
---
Abschluss 2020 mit Bestnote in Web & Ui-Design
Kommentar der IHK war, dass die Webseite so schön ist, dass man lieber auf Ihr bleibt, statt zum Optiker zu gehen :D
